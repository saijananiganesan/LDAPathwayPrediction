Identify similar metabolic pathways and enzyme clusters using latent Dirichlet allocation (LDA)

About

This application is limited to identifying co-occurring enzyme clusters in prokaryotes. Metabolic network data mined from KEGG database is used as a document, enzyme classification numbers are used as words to identify hidden co-occurring enzyme clusters (CEC) as topics. For details on the method, refer the code repository

Usage

There are three options to use this application:

  1. Identify topics (or CEC) given a list of enzyme classification numbers.
  2. Identify similar pathways using LDA given a list of enzyme classification numbers.
  3. Identify similar pathways using FastText algorithm given a list of enzyme classification numbers.

Choosing the right application of interest

  1. LDA: To be chosen if the requirement is to (a) get topics (or CEC) given a list of enzymes and (b) probabilities associated with outcomes.
  2. FastText: To be chosen if the input list contains enzymes that do not frequently occur in prokaryotes.

Note: FastText works by considering similarity between words (or characters). For example, EC.1.2.3.4 will be characterized as similar to EC.1.2.3.98 by the algorithm even if EC.1.2.3.4 was never observed during training. Whereas LDA will simply assign a zero probability to enzymes that the model has never seen before.
